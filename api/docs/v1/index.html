Book of Changes API v1 Documentation

Book of Changes API v1 is a read-only repository for fetching I Ching data.

Hexagrams

The Hexagrams endpoint

/api/v1/hexagrams

Arguments:

  • id —Required: accepts integers 1 through 64 or any six-digit cobmination of 1 and 0

    Example: both id=011110 and id=28 will return results for "Great Exceeding | Great Preponderance | 'Potential'"

  • moving_lines —Optional: accepts a combination of integers 1 through 6 no more than six digits

    Example: passing moving_lines=345 will return messages for the respective lines of the requested Hexagram; omitting this argument will return the 'still lines' message.

  • dual —Optional: accepts true or false

    Passing dual=true along with a moving lines argument will return two Hexagrams: the Hexagram requested by the id argument and the Hexagram formed by applying the moving lines to requested Hexagram's lines.

  • verbose —Optional: accepts true or false

    Passing verbose=true will return additional fields: inner and innermost Hexagrams, upper and lower Trigrams, and special unicode characters.

The Hexagrams-filtered-by-Trigram endpoint

/api/v1/hexagrams/trigram

Arguments:

  • segment —Required: accepts 'top' or 'bottom'

  • id —Required: accepts any three-digit cobmination of 1 and 0

Example: the query ?id=101&segment=top will return all Hexagrams with "Fire" as their upper Trigram.

The Inner Hexagrams endpoint

/api/v1/hexagrams/inner

Returns all Inner Hexagrams

The Sovereign Hexagrams endpoint

/api/v1/hexagrams/sovereign

Returns all Sovereign Hexagrams

Trigrams

The Trigrams endpoint

/api/v1/trigrams

Arguments:

  • id —Required: accepts any three-digit cobmination of 1 and 0

    Example: the query ?id=101 will return the "Fire" Trigram.
